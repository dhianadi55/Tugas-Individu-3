Responsive Layout

  • ITERA Institut Teknologi Sumatera
  • JTPI Jurusan Teknologi Produksi dan Industri
  • IF21 Informatika 21

Membuat desain responsive halaman, dengan kriteria:

  1. Minimal ada 3 layout yang ditampilkan
  2. Ada 3 variasi ukuran layar: 1 ukuran handphone, tablet, pc/laptop
  3. Gunakan minimal 3 contoh elemen block dan 3 elemen inline pada desain halaman web
  4. Buat salah satu layout menggunakan properti flexbox parent
